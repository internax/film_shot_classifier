\doxysubsection{Detailed Description}
Evaluates classification accuracy against a labeled test dataset. 

The {\ttfamily \doxylink{class_test_dataset_eval}{Test\+Dataset\+Eval}} class is designed to compare predicted classification results with a predefined set of ground truth labels. It supports loading the ground truth data, adding prediction samples, and computing an evaluation score (e.\+g., accuracy).

This is useful for testing and validating the performance of the shot classifier on labeled datasets.

Example usage\+: 
\begin{DoxyCode}{0}

\DoxyCodeLine{evaluator.loadGroundTruth(\textcolor{stringliteral}{"{}ground\_truth.csv"{}});}
