\doxysubsection{Detailed Description}
Classifies the type of cinematic shot based on extracted visual features. 

\doxylink{class_shot_classifier}{Shot\+Classifier} takes a \doxylink{struct_shot_features}{Shot\+Features} input and produces a classification result. Typically used after feature extraction from a single video frame.

The classification logic is based on heuristics, such as the relative size of detected objects compared to the frame.
